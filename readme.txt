Program created by Jack Page and Runlong Zhou
This program is written as an example of a lexer for CPSC 323 at Cal State Fullerton

It contains 6 files;
FSA_mydesign.docx, a document explaining the design of our FSA and call structure
input_scode.txt, a txt doc containg the code provided by the corse to analyze 
lexar.cpp, c++ code for running our lexar 
lexar.h, a header file containing the functions called in lexar.cpp
output.txt, a place to store the lexars output
readme.txt, this file :)

To run file 
g++ -o <lexar> lexar.cpp
./lexar
or prefered running method
